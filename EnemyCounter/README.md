# Enemy Counter

Displays the number of currently alive enemies in the info panel (top-left corner of the screen).

## Counter Colors

| Color    | Meaning                       |
| -------- | ----------------------------- |
| Red      | At least one enemy is alive.  |
| Orange   | Enemies can spawn.            |
| Yellow   | Only bosses can spawn.        |
| Green    | No more enemies can spawn.    |
